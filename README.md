# Gestión de Cuentas de Cobro y Cotizaciones

Aplicación web MVC en PHP con AdminLTE 4 para gestionar clientes, cotizaciones y cuentas de cobro.

## Requisitos
- PHP 8.1+
- Extensión PDO (MySQL)
- Servidor web con soporte para `.htaccess`
- MySQL/MariaDB

## Instalación
1. Subir todos los archivos al directorio público.
2. Copiar `adminlte/dist/` (ya incluido) en `adminlte/`.
3. Configurar credenciales en `config/BaseDeDatos.php`.
4. Importar `database/esquema.sql` en la base de datos.
5. Acceder a la URL y autenticarse con `admin@demo.com` / `Admin123*`.

## Créditos
- AdminLTE 4 (https://adminlte.io)
- Bootstrap 5, jQuery y DataTables vía CDN.
